قم ببناء تطبيق ويب احترافي لتتبع طلبيات الزبائن لشركة تعمل كوسيط شراء من الإنترنت.

شرط أساسي:
- يجب أن تكون لغة التطبيق عربية بالكامل 100٪
- جميع النصوص، الأزرار، الرسائل، العناوين، والتنبيهات باللغة العربية فقط
- اتجاه الواجهة من اليمين إلى اليسار (RTL)

أنواع المستخدمين:
1. الزبون (بدون تسجيل دخول)
2. الإدارة (تسجيل دخول إلزامي)

واجهة الزبون:
- صفحة لتتبع الطلبية باستخدام:
  - رقم الطلبية
  - أو رقم الهاتف
- بعد البحث يتم عرض:
  - رقم الطلبية
  - حالة الطلب الحالية
  - شريط يوضح مراحل الطلب
  - تاريخ الوصول المتوقع
  - ملاحظات الإدارة (إن وُجدت)
- يمنع عرض أي معلومات عن الموردين أو المتاجر

مراحل حالة الطلب (ثابتة):
1. تم استلام الطلب
2. تم تأكيد الدفع
3. تم الشراء من الموقع
4. قيد الشحن من المصدر
5. وصلت إلى بلد العبور
6. وصلت إلى ليبيا
7. قيد التوصيل
8. تم التسليم
9. ملغاة / توجد مشكلة

واجهة الإدارة:
- صفحة تسجيل دخول آمنة (اسم مستخدم + كلمة مرور)
- لوحة تحكم تحتوي على:
  - إضافة طلبية جديدة
  - تعديل بيانات الطلبية
  - تغيير حالة الطلب من قائمة منسدلة
  - تحديد تاريخ الوصول المتوقع
  - إضافة ملاحظات للزبون
  - البحث برقم الطلبية أو رقم الهاتف
  - حذف الطلبية عند الحاجة

بيانات الطلبية (MySQL Table):
- id (مفتاح أساسي)
- order_number (فريد)
- customer_name
- phone_number
- order_status
- estimated_delivery_date
- admin_notes
- created_at
- updated_at

بيانات المستخدم الإداري:
- id
- username
- password (مشفر)

متطلبات التصميم:
- تصميم بسيط واحترافي
- متوافق مع الموبايل
- شريط مراحل واضح
- خطوط عربية
- دعم كامل لـ RTL

المتطلبات التقنية:
- Frontend: HTML + CSS + JavaScript
- Backend: Node.js + Express
- Database: MySQL
- استخدام mysql2 أو sequelize للاتصال بقاعدة البيانات
- تخزين بيانات الاتصال بقاعدة البيانات في ملف بيئة (.env)
- حماية صفحة الإدارة بتسجيل دخول
- الزبون لا يحتاج إلى حساب

ميزات إضافية (اختيارية):
- رابط تتبع خاص لكل طلبية
- رسائل توضيحية عند التأخير
- تجهيز التطبيق للتحويل إلى تطبيق موبايل لاحقاً

هدف المشروع:
تقليل الاتصالات الهاتفية المتكررة عبر تمكين الزبائن من تتبع طلبياتهم بأنفسهم، مع تحكم كامل للإدارة في تحديث الحالات.
